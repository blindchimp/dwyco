FL_version

Section: User Manuals (3)
Updated: Dec 2000

NAME

FL_version - returns library version information.  

SYNOPSIS

#include <fl.h>

void FL_version(int *major_ver, int *minor_ver, int *patch_ver);  

DESCRIPTION

FL_version returns the major, minor, and patch version numbers of the Flush Spread library in use.
